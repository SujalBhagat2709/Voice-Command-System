# Voice Command System

## Overview
A simple AI-style voice command system using speech recognition.

---

## Features
- Listen to voice input
- Convert speech to text
- Execute simple commands

---

## Files
- `listen.py` → captures voice input
- `commands.py` → processes commands

---

## Installation

```bash
pip install SpeechRecognition pyaudio
```

---

## Usage

```bash
python commands.py
```

---

## Example Commands

```text
hello
time
exit
```

---

## Example Output

```text
You said: hello
Response: Hello! How can I help you?
```
